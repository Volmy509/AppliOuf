# OufCorp - Système de Gestion

## Description
Application Java pour gérer des cartes d'employés et des achats de produits.

## Fonctionnalités
- Gestion des employés
- Gestion des cartes (Standard/Gold)
- Gestion des produits
- Système d'achats
- Consultation de solde

## Installation

### Prérequis
- Java 8 ou supérieur

### Lancer le programme
```bash
java -jar OufCorp.jar
```
